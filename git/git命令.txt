git remote 列出已经存在的远程分支

git remote -v 列出详细信息

git remote add 别名 url远程库地址

git fetch 别名  抓取所有远程仓库信息（本地没有的）

-------------------------------------------------------------
git config --global（local） user.name "用户名"

git config--global （local）user.email "邮箱地址"

git init 初始化一个远程仓库

-----------------------------------------------------------------

git status  查询未提交暂存区的文件

git add 文件名    将文件提交到暂存区

git add .  提交被修改和新建的文件，但不包括被删除的文件

git add -u --update  更新所有改变的文件，即提交所有变化的文件

git add -A --all   提交已被修改和已被删除的文件，但不包括新的文件

git commit -m "索引信息"  文件名  提交到本地仓库

git commit -a -m “索引信息”

-------------------------------------------------------------
版本回退：

git log    查看最近提交历史，查看版本库信息

git log --pretty=oneline     查看提交历史简化

git reset --hard HEAD^   回退到上一个版本

git rest --hard HEAD^^  回退到上上个版本

git rest --hard 版本号（写一部分就ok，比如21a77） 回退到某个版本

git reflog  查看命令历史

-------------------------------------------------------------
撤销：

git checkout   文件名    未通过add命令到缓存区之前，可撤销

git diff 查看暂存区和工作区有什么不同  已经添加到缓存区 需要 git rest撤销

git rm 文件名  删除文件(先删除工作区，再删除暂存区)

git rm -f 文件名   #同时删除工作区和暂存区的文件

git rm --cached 文件名   删除暂存区文件，不删除工作区文件
-------------------------------------------------------------
git  push origin master        --库名 --分支名 将本地分支跟踪到远程分支

--------------------------------------------------------------
设置sshkeys：

ssh-keygen -t rsa -C  自己的邮箱      一路回车后在c盘user中找.ssh文件夹，选择id_rsa.pub 里的内容复制到远程仓库的ssh keys中

---------------------------------------------------------------------
git 删除本地仓库步骤

1、git branch  #显示所有本地分支

2、git init   #初始化本地版本库

3、ls -a  #找到目录下的.git

4、rm -rf .git   #删除

---------------------------------------------------------------------
分支管理：

git checkout -b 分支名  #创建一个分支

git checkout 分支名 #切换分支

git branch  查看所有分支  当前分支前有*

git merge 分支名   合并分支名到当前分支
